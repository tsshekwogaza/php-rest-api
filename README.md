# PHP REST API

A simple REST API built with PHP (LAMP stack).

## Features
- GET, POST, PUT, DELETE
- MySQL integration
- JSON responses
- Routing system

## Endpoints

GET /api/users  
POST /api/users  
PUT /api/users/{id}  
DELETE /api/users/{id}

## Tools
- PHP
- MySQL
- Apache
